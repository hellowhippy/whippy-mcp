🚀 Whippy MCP Server

Model Context Protocol (MCP) Server for Whippy AI

This server provides AI assistants with access to Whippy AI's powerful messaging and contact management capabilities through the Model Context Protocol.

📱 Available Tools

  • Contact Management Create, retrieve, and manage contacts
  • Messaging Send SMS and email messages
  • Campaigns Create and manage marketing campaigns
  • Leads Track and manage leads
  • Analytics Get campaign performance metrics

⚙️ Setup Instructions

To use this MCP server with your AI assistant:

  1. Get your Whippy API key from whippy.ai
  2. Configure your AI client to connect to this server
  3. Start using Whippy's messaging and contact management tools!
